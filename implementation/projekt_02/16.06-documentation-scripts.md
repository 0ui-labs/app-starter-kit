### 4. GitHub Actions Workflow
**.github/workflows/docs.yml:**
```yaml
name: Documentation
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  build-docs:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
      - run: npm ci
      - run: npm run docs:build
      - uses: actions/upload-artifact@v3
        with:
          name: documentation
          path: docs/
```
