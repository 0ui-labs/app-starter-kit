# Teilaufgabe 10.06: Build Test durchführen

## Ziel
Verifizieren, dass das Projekt nach React 19 Update erfolgreich baut.

## Warum Build Test?
- Stellt sicher, dass alle Packages kompatibel sind
- Findet Type Mismatches zwischen Packages
- Verifiziert, dass React 19 korrekt funktioniert

## Implementierung

### Schritt 1: Build ausführen
```bash
npm run build
```

### Schritt 2: Build Output analysieren
Achte auf:
- React Version Warnings
- Type Mismatch Errors
- Missing Dependencies
- JSX Transform Warnings

### Schritt 3: Bei Fehlern - Detailliertes Build
```bash
# Für mehr Details
npm run build -- --verbose

# Nur bestimmtes Package bauen
npm run build -- --filter=@starter-kit/ui
```

## Erwartete Ausgabe
```
✓ Packages built successfully
✓ No React version warnings
✓ No type mismatches
```

## Mögliche Fehler und Lösungen

### Fehler: React Version Mismatch
```
Error: React version mismatch between packages
```
**Lösung**: Prüfe alle package.json auf konsistente React Versionen

### Fehler: Type Errors
```
Type error: Property 'children' does not exist
```
**Lösung**: React 19 benötigt explizite children Definition:
```tsx
interface Props {
  children?: React.ReactNode;
}
```

### Fehler: JSX Transform
```
Error: JSX element implicitly has type 'any'
```
**Lösung**: Stelle sicher dass `"jsx": "react-jsx"` in tsconfig.json

### Fehler: Module Resolution
```
Cannot find module '@starter-kit/ui'
```
**Lösung**: 
```bash
# Turbo Cache löschen
rm -rf .turbo
rm -rf apps/*/.turbo packages/*/.turbo
npm run build
```
