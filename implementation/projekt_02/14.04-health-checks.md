### 3. Response Format
```json
{
  "status": "healthy|degraded|unhealthy",
  "timestamp": "2025-01-05T10:00:00Z",
  "duration": 50,
  "checks": [
    {
      "service": "database",
      "status": "healthy",
      "responseTime": 15
    }
  ],
  "version": "1.0.0",
  "environment": "production"
}
```

### 4. Status Codes
- 200: All services healthy
- 503: One or more services unhealthy
- Degraded services still return 200
